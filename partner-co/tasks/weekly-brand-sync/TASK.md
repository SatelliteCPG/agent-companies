# Weekly Brand Sync

1. Pull recent activity for each brand using search_opportunities and list_meetings.
2. Summarize pipeline changes — new opportunities, stage advances, wins, losses.
3. Flag brands with stalled pipeline, missed meetings, or unresolved deductions.
4. Build 2-3 talking points per brand for sync calls.
5. Identify brands that have not been contacted this month and flag for outreach.
6. Compile sync materials and distribute to VP Client Services for review.
