# Weekly Supplier Onboarding Status

1. Count applications at each pipeline stage (Application, Evaluation, Compliance, Setup, Go-Live).
2. Identify any supplier stalled at a stage for more than 14 days. Document the blocker.
3. List suppliers expected to go live in the next 30 days with projected dates.
4. Note any compliance issues delaying onboarding (GS1, EDI, insurance, food safety).
5. Report new applications received in the past week with initial category assessment.
6. Present pipeline summary to Director of Supplier Relations.
