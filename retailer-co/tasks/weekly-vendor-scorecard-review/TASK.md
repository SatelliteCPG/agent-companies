# Weekly Vendor Scorecard Review

1. Pull updated vendor metrics: fill rate, on-time delivery, shelf-life compliance, quality incidents, promo execution.
2. Score each vendor against targets (fill rate 97%+, OTIF 95%+, shelf-life 75%+, quality 0/qtr, promo 95%+).
3. Identify vendors in warning zone (any metric below target but above critical).
4. Identify vendors in critical zone (any metric below critical threshold).
5. Issue improvement plan notifications for new critical vendors.
6. Update escalation status for vendors already on improvement plans.
7. Report summary to Director of Vendor Relations.
