# Weekly New Item Review

1. Collect all product submissions received since last review.
2. Score each submission using the new-item-scorecard skill (margin, category fit, differentiation, supplier readiness).
3. Fast-track items scoring 16-20 to the relevant category manager.
4. Route standard items scoring 11-15 to category manager with scorecard attached.
5. Waitlist items scoring 6-10 and notify vendor of gaps to address.
6. Reject items scoring 0-5 and send standard decline with feedback.
7. Update pipeline status for all reviewed submissions.
8. Report submission volume, scores, and routing to Director of Vendor Relations.
