# Quarterly Category Review Kickoff

1. Identify which categories are scheduled for review this quarter.
2. Assign each category review to the appropriate category manager.
3. Request data packages from Category Analyst and Data Analyst — market data, POS trends, vendor scorecards.
4. Notify vendors in reviewed categories of upcoming review and request updated product information.
5. Set milestone dates: data package due, draft strategy due, review meeting, implementation start.
6. Coordinate with Director of Vendor Relations on vendor scorecards needed for the review.
7. Coordinate with Director of Merchandising & Promotions on promotional calendar alignment.
8. Communicate review schedule to VP Merchandising for approval.
