# Weekly Trade Spend Reconciliation

1. Pull all new deductions from the past week.
2. Classify each deduction: valid (promotional), valid (damage/shortage), or invalid.
3. File disputes on invalid deductions with supporting documentation.
4. Update the deduction aging report with current balances and statuses.
5. Summarize weekly recovery rate and outstanding balance for vp-finance.
