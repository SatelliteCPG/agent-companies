# Weekly SPINS Review

1. Pull latest weekly SPINS data for all tracked categories and retailers.
2. Compare velocity vs prior 4-week average and year-ago period.
3. Benchmark against top 5 competitors in each category.
4. Flag any distribution gains or losses (new stores, lost stores).
5. Summarize key takeaways and share with sales and marketing teams.
