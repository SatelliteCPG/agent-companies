# Monthly Category Review Prep

1. Identify upcoming retailer category review windows for the next 60 days.
2. Compile latest SPINS data and trend analysis for each target category.
3. Build competitive analysis showing share, velocity, and distribution gaps.
4. Summarize promotional performance from prior period.
5. Prepare new item proposals with projected incrementality.
6. Package into retailer-specific submission format.
